Welcome to your School CMS

Use the sidebar to configure everything visible on the public site. A few good places to start:

  • Site Settings — change the title, address, contact details and the welcome paragraph.
  • Menu Manager — add, rename or hide menu items, and link each one to a page.
  • Popular Links — manage the external link cards in the Most Popular Links block.
  • News & Events — publish the news items shown on the home page.

Database connection

The MySQL connection is read from the environment variable DATABASE_DSN (with DATABASE_USER / DATABASE_PASSWORD). Never hard-code credentials in source files. See .env.example.

These values are surfaced on every public page (title, address, contact info, footer text, etc.).

Existing menu items

Each menu item maps a Label to a page (internal slug) or external URL. Toggle Public off to hide an item without deleting it.
